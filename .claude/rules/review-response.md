# @claude レビュー対応ルール

GitHub 上で `@claude` とメンションされた際の対応方針。

## 対応の判断基準

| コメントの種類 | 判断基準 | 対応 |
|--------------|---------|------|
| 質問 | 「なぜ？」「これ何？」「教えて」など | コメントで回答のみ（コード変更しない） |
| 修正依頼 | 「直して」「変えて」「追加して」など | コード修正 + 何を修正したかコメントで報告 |
| 曖昧 | 「ここ気になる」「これどうなの」など | まずコメントで意図を確認してから対応 |

## 対応時の注意

- 修正した場合は**必ず何を変えたかコメントで報告する**（修正だけして無言にしない）
- 修正の範囲はコメントで指摘された箇所に限定する（余計な変更をしない）
- 判断に迷う場合は修正せずコメントで確認を取る
