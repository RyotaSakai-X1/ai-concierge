# /briefing — 今日のブリーフィング

今日のスケジュールを確認して、わかりやすく報告してください。

## 手順

### 1. 今日の予定を取得

Google Calendar から今日の予定を取得してください。

### 2. 予定を一覧で表示

以下の形式で報告してください：

```
📅 本日の予定（YYYY年MM月DD日）

- 10:00〜11:00 ○○ミーティング（参加者: △△、□□）
- 14:00〜15:00 △△定例
- 終日予定なし の場合は「本日の予定はありません」
```

### 3. 関連情報があれば添える

各予定について、以下に該当するものがあれば添えてください：

- 予定のタイトルや説明文に **イシュー番号（#XX）** が含まれていれば、`gh issue view` でイシューの状態を取得して添える
- 予定のタイトルや説明文から推測できる **関連する仕様書やドキュメント** が `docs/` 配下にあれば、ファイルパスを添える
- 特に関連情報がなければ無理に探さない（シンプルに予定だけ表示）

### 4. 注意事項の補足

- 直近1時間以内に始まる予定があれば「まもなく開始」と目立つように伝える
- 予定が詰まっている場合は「本日は予定が多いです」と一言添える
