# ビルトインツール一覧

> 公式ドキュメント + リリースノート（v2.1.16〜v2.1.74）から抽出。最終更新: 2026-03-12

| ツール | 説明 |
|-------|------|
| `Bash` | シェルコマンド実行 |
| `Read` | ファイル読込（PDF pages指定対応） |
| `Edit` | ファイル編集（差分ベース） |
| `Write` | ファイル作成・上書き |
| `Glob` | ファイルパターン検索 |
| `Grep` | コンテンツ検索（ripgrep） |
| `Agent` | サブエージェント起動（model指定可） |
| `TaskCreate` | タスク作成 |
| `TaskGet` | タスク取得 |
| `TaskList` | タスク一覧 |
| `TaskUpdate` | タスク更新 |
| `TaskOutput` | バックグラウンドタスク出力取得 |
| `TaskStop` | タスク停止 |
| `LSP` | 定義ジャンプ・参照検索 |
| `EnterWorktree` | ワークツリー開始 |
| `ExitWorktree` | ワークツリー終了 |
| `CronCreate` | cron定期実行作成 |
| `CronDelete` | cron定期実行削除 |
| `CronList` | cron定期実行一覧 |
| `WebSearch` | Web検索 |
| `WebFetch` | URL取得・解析 |
| `NotebookEdit` | Jupyterノートブック編集 |
| `AskUserQuestion` | ユーザーへの質問（選択UI） |
| `ToolSearch` | 遅延ロードツールの検索・取得 |
| `EnterPlanMode` | プランモード開始 |
| `ExitPlanMode` | プランモード終了 |
