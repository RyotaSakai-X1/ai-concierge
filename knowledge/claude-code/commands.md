# スラッシュコマンド一覧

> 公式ドキュメント + リリースノート（v2.1.16〜v2.1.74）から抽出。最終更新: 2026-03-12

| コマンド | 説明 |
|---------|------|
| `/add-dir <path>` | 追加ワーキングディレクトリ指定 |
| `/agents` | エージェント設定管理 |
| `/btw <question>` | 会話履歴に残さないサイド質問 |
| `/chrome` | Chrome連携設定 |
| `/clear` | 会話履歴クリア（`/reset`, `/new`） |
| `/compact [instructions]` | コンテキスト圧縮（フォーカス指示可） |
| `/config` | 設定画面（`/settings`） |
| `/context` | コンテキスト使用量確認・最適化提案 |
| `/copy` | 最後の応答をコピー（コードブロック選択可、`w`でファイル書出し） |
| `/color` | プロンプト色変更（`default`でリセット） |
| `/cost` | トークン使用統計 |
| `/desktop` | Desktopアプリで続行（`/app`） |
| `/diff` | 未コミット変更の対話的diff表示 |
| `/doctor` | 環境診断・設定検証 |
| `/effort` | 推論努力度切替（low/medium/high/auto） |
| `/exit` | CLI終了（`/quit`） |
| `/export [filename]` | 会話をテキストエクスポート |
| `/extra-usage` | 追加使用量設定 |
| `/fast [on\|off]` | Fast Mode切替 |
| `/feedback [report]` | フィードバック送信（`/bug`） |
| `/fork [name]` | セッション分岐 |
| `/help` | ヘルプ・コマンド一覧 |
| `/hooks` | フック設定管理 |
| `/ide` | IDE連携管理 |
| `/init` | CLAUDE.md初期化 |
| `/insights` | セッション分析レポート生成 |
| `/install-github-app` | GitHub Actions連携セットアップ |
| `/install-slack-app` | Slackアプリインストール |
| `/keybindings` | キーバインド設定ファイルを開く |
| `/login` | ログイン |
| `/logout` | ログアウト |
| `/loop` | 定期実行（例: `/loop 5m /briefing`） |
| `/mcp` | MCPサーバー管理 |
| `/memory` | CLAUDE.mdメモリ編集・自動メモリ管理 |
| `/mobile` | モバイルアプリQRコード表示（`/ios`, `/android`） |
| `/model [model]` | モデル切替（処理中でも可、左右キーで努力度調整） |
| `/passes` | Claude Code無料パス共有 |
| `/permissions` | パーミッション管理（`/allowed-tools`） |
| `/plan [description]` | プランモード（引数で即時開始可） |
| `/plugin` | プラグイン管理・マーケットプレイス |
| `/pr-comments [PR]` | GitHub PRコメント取得・表示 |
| `/privacy-settings` | プライバシー設定 |
| `/release-notes` | リリースノート表示 |
| `/reload-plugins` | プラグイン再読込 |
| `/remote-control` | リモート操作セッション開始（`/rc`） |
| `/remote-env` | テレポート用リモート環境設定 |
| `/rename [name]` | セッション名変更（自動生成対応） |
| `/resume [session]` | セッション再開（`/continue`） |
| `/rewind` | 会話・コードを巻き戻し（`/checkpoint`） |
| `/sandbox` | サンドボックス切替 |
| `/security-review` | セキュリティレビュー |
| `/skills` | スキル一覧表示 |
| `/stats` | 使用統計・セッション履歴 |
| `/status` | バージョン・モデル・接続状態表示 |
| `/statusline` | ステータスライン設定 |
| `/stickers` | ステッカー注文 |
| `/tasks` | バックグラウンドタスク管理 |
| `/terminal-setup` | ターミナルキーバインド設定 |
| `/theme` | テーマ・構文ハイライト切替 |
| `/upgrade` | プランアップグレード |
| `/usage` | プラン使用量・レート制限表示 |
| `/vim` | Vim編集モード切替 |
