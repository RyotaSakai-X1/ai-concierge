# デザインリファレンス集

案件で見つけた参考サイト・UI パターン・デザインリソースを蓄積する場所。

## ファイル構成

| ファイル | 内容 |
|---------|------|
| `sites.md` | 参考サイト集（URL + 参考要素 + カテゴリ） |
| `patterns.md` | 良い UI パターンの収集（スクリーンショット・説明付き） |
| `resources.md` | デザインツール・ライブラリ・アセットのリンク集 |

## 追加ルール

- 新しいサイトを見つけたら `sites.md` に追加する
- 必ず「何を参考にするか」を具体的に書く（URL だけ貼らない）
- カテゴリタグを付けて検索しやすくする
