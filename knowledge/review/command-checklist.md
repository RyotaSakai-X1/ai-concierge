# コマンド定義レビューチェックリスト

`.claude/commands/*.md` のレビュー観点。

## チェック項目

| カテゴリ | チェック項目 |
|---------|------------|
| 構造 | Step 構造（Step N: タイトル）の有無 |
| 構造 | 各 Step に入力・処理ルール・出力の明示 |
| 宣言的スタイル | 手続的表現（「〜があるか」「〜しているか」「〜する」の連続）の不在 |
| 宣言的スタイル | テーブル・箇条書き中心の構成（長文説明の不在） |
| 宣言的スタイル | 分岐条件がテーブルで表現されている |
| 精度 | 分析観点にアクション動詞+具体キーワードが含まれる |
| 精度 | AskUserQuestion の選択肢が具体的に列挙されている |
| 精度 | エラーケースの対応が記載されている |
| 一貫性 | 引数セクションの形式統一 |
| 一貫性 | 制約セクションの存在 |
