# ナレッジ一覧

AI がコマンドを実行するときに参照する「知識」をまとめたフォルダです。
ここにルールや過去の事例を追加すると、AI の判断がより正確になります。

## 現在のナレッジ

| 分野 | フォルダ | 内容 |
|------|---------|------|
| 共通 | `knowledge/business-rules.md` | 自社ルール・制約・前提知識 |
| テンプレート | `knowledge/templates/spec.md` | 仕様書テンプレート |
| テンプレート | `knowledge/templates/report.md` | レポートテンプレート |
| テンプレート | `knowledge/templates/review-report.md` | レビュー結果テンプレート |
| レビュー | `knowledge/review/spec-checklist.md` | 仕様書レビューの観点 |
| レビュー | `knowledge/review/output-checklist.md` | 成果物レビューの観点 |

## ナレッジの追加方法

該当する分野のフォルダに Markdown ファイルを追加してください。

```
例：見積もりのルールを追加する場合
knowledge/estimation/guidelines.md に記述
```

既存のフォルダがない分野の場合は、新しいフォルダを作成してください。
