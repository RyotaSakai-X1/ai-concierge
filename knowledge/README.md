# ナレッジ一覧

AI がコマンドを実行するときに参照する「知識」をまとめたフォルダです。
ここにルールや過去の事例を追加すると、AI の判断がより正確になります。

## 現在のナレッジ

| 分野 | フォルダ | 内容 |
|------|---------|------|
| （まだありません。コマンド追加時にここも更新されます） | - | - |

## ナレッジの追加方法

該当する分野のフォルダに Markdown ファイルを追加してください。

```
例：見積もりのルールを追加する場合
knowledge/estimation/guidelines.md に記述
```

既存のフォルダがない分野の場合は、新しいフォルダを作成してください。
